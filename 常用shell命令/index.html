    常用shell命令

    批量文件重命名

    # 把.txt 后缀的改成 .java后缀
rename 's/.txt/.java/' *.txt

    显示当前目录下文件的大小

    ls -lh

    显示当前目录的大小

    du -sh

    查找文件

    # 查找当前目录及其子目录下的txt文件
find . -name '*.txt'

    显示当前目录文件数

    ls |wc -l

    显示标准进程信息

    ps -ef

    让进程在后台运行

    # 新建窗口并取个名字
screen -S test
# 在新窗口中运行程序
java -jar test.jar
# 在当前会话窗口中按Ctrl a + d快捷键可以实现分离，并回到主窗口
# 列出窗口
screen -ls
# 恢复会话窗口
screen -r 1234
# 杀死会话窗口
kill -9 1234
# 清除死去的窗口
screen -wipe

    查看端口信息

    netstat -ntlp|grep 8888

    查看某文件夹下文件和文件夹的个数

    ls -l |wc -l

    查看某文件夹下文件的个数

    ls -l |grep "^-"|wc -l

    查看某文件夹下文件夹的个数

    ls -l |grep "^d"|wc -l

    查看某文件夹下文件的个数，包括子文件夹里的:

    ls -lR|grep "^-"|wc -l

    查看某文件夹下文件夹的个数，包括子文件夹里的:

    ls -lR|grep "^d"|wc -l
