    Android5.0之后禁止隐式声明Intent启动Service

    使用显示声明Intent

    Intent intent = new Intent();
//方式1
ComponentName componentName = new ComponentName(packageName, serviceName);
intent.setComponent(componentName);
//方式2
intent.setPackage(packageName);

    依赖aar

    在module的build.gradle中添加如下：

    repositories {
    flatDir {
        dirs 'libs'
    }
}
dependencies {
    implementation(name: 'libname', ext: 'aar')
}

    解决INSTALL_FAILED_TEST_ONLY

    在gradle.properties文件中添加android.injected.testOnly=false

    跳转到Google Play应用界面

    Intent intent = new Intent();
intent.setAction(Intent.ACTION_VIEW);
intent.setData(Uri.parse("market://details?id=" + objectPackageName));
if (intent.resolveActivity(context.getPackageManager()) != null) {
    context.startActivity(intent);
} else {
    intent.setData(Uri.parse("https://play.google.com/store/apps/details?id=" + objectPackageName));
    context.startActivity(intent);
}

    Android Studio多个project依赖同一module

    在settings.gradle中添加

    include 'android-common'
project(':android-common').projectDir = new File('../android-common')
include ':android-common:commonlib'
